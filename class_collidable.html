void Collidable::CollisionRegistration ( C *  c)
inlineprotected

This function registers the game object desired for collisions.

Note
This must be done in order for users to be able to create a collision test with a GameObject.

Example This is an example of how the user can register a cottage object as a collidable.

void Cottage::SceneEntry()
{
CollisionRegistration<Cottage>(this);
}

c The GameObject the user wants to register for collision
