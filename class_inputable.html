void Inputable::KeyPressed ( AZUL_KEY  k)
inlineprivatevirtual

This function is to be overridden by the user whenever a key pressed is registered.

Example This is an example of how the KeyPressed function can be overriden within a user GameObject.

void PlayerTank::KeyPressed(AZUL_KEY k)
{
switch (k)
{
case AZUL_KEY::KEY_SPACE:
DebugMsg::out("<SPACE> pressed \n", k);
FireTankShell();
break;
}
}

k The key that is being pressed.
