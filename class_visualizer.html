inlinestatic

This function renders the BSphere.

Note
The ShowBSphere function MUST be in the draw() function of a gameobject.

Example This is an exmaple of the EnemyTank rendering a bounding sphere and changing the color based upon whether or not there a collision.

void EnemyTank::Draw()
{
pTank->Render(GetCurrentCamera());
if (collide == false)
{
Visualizer::ShowBSphere(GetBSphere(), Colors::White);
}
else
{
Visualizer::ShowBSphere(GetBSphere(), Colors::Red);
}
collide = false;
}
